# FreeKassa.COM
> Неофициальная реализация интеграции к сервису [FreeKassa.com](https://FreeKassa.com "Перейти на сайт FreeKassa.com")

## О сервисе
FreeKassa — это система онлайн-платежей, которая позволяет интернет-магазинам и другим коммерческим ресурсам принимать платежи через различные методы, включая карты и электронные деньги. Она предлагает более ста способов оплаты и подходит как для частных лиц, так и для бизнеса. ### Основные характеристики FreeKassa

Типы регистрации:

Мерчант-касса: Позволяет принимать платежи на сайте физического лица. Вывод средств доступен сразу.
Компания: Для юридических лиц, требует дополнительного одобрения, но комиссии ниже.
Кошелек: Позволяет физическим лицам делать платежи и массовые выплаты.
Процесс подключения:

Регистрация проекта в системе FreeKassa.
Получение идентификатора проекта и генерация секретных слов.
Настройка URL для уведомлений и успешных/неуспешных платежей.
Поддерживаемые методы оплаты:

Платежи по картам, электронными деньгами, Apple Pay и другими способами.
Преимущества использования FreeKassa
Широкий выбор методов оплаты: Более ста способов, что позволяет удовлетворить потребности различных клиентов.

Удобство для бизнеса: Быстрый вывод средств и возможность интеграции с различными платформами.

Поддержка и документация: Наличие техподдержки и подробных инструкций по настройке.
